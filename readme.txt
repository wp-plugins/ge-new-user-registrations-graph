A dashboard widget to show a graph of new user registrations per day

== Description ==

This plugin provides a graph of new user registrations in the admin dashboard. You can choose to view this graph over a time period of a week, two weeks, 30 days, 90 days, or a year.

== Installation ==

Upload the plugin
Go to the Admin Dashboard. 
At the top of the page, click on “Screen Options”. 
You should see an option called “New User Registrations”. Make sure you put a little tick next to it.
